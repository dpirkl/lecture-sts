# Lecture STS

This repository can be used to translate any video to english. It uses OpenAI's whisper to transcribe and translate the video. Then it uses coqui AI's TTS to synthesize the transcription. Finally, it combines captions and the translated audio with the video.

It originally was build to translate the lecture 'Introduction to AI' in the winter term of 2022/23 at Technische Universität Darmstadt.
